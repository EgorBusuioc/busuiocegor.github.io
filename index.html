    Project Name: EasyLease
    Technology Stack:
    Java, Spring Boot, Multiple PostgreSQL instances, Apache Kafka, Spring Security, JWT, Email confirmation, Eureka, API Gateway.
    Implemented Microservices:
    Gateway Service:
    Routes incoming client requests to the appropriate backend services.
    Auth Service:

    Notification Service:
    Sends email notifications, including activation links.

    Manages apartments and works with a separate database. Communicates securely via API key with the Auth Service.
